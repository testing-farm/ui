= Testing Farm UI

A web UI for Testing Farm.

== Installation

NOTE: You might need to install the `libffi-devel` package first.

[source,bash]
....
poetry install
....

== Testing

[source,bash]
....
poetry run tox
....

== Running Locally

[source,bash]
....
ENV_FOR_DYNACONF=development poetry run reflex run
....

== Mock Logging

The `development` dynaconf environment enables mock login for easier local testing.
